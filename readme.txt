📌 Sobre o Projeto

Este repositório contém uma solução em Python desenvolvida para automatizar a coleta e o tratamento de dados operacionais. O objetivo é reduzir o tempo de geração de relatórios e aumentar a precisão na análise de indicadores técnicos.
🛠 Tecnologias Utilizadas

    Python: Processamento e lógica principal.

    Pandas/NumPy: Manipulação e limpeza de dados.

    Matplotlib/Seaborn: Visualização de dados.

📊 Visualização de Dados: O Poder do Heatmap

O projeto encerra com a geração de um mapa de calor. Esta escolha técnica visa:

    Intuição Visual: Identificar rapidamente picos e anomalias.

    Suporte à Decisão: Facilitar a leitura para equipes de campo e gestores que precisam de respostas rápidas.

    Padronização: Converter dados complexos em uma matriz de cores fácil de digerir.
